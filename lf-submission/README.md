# LF AI & Data Submission Package — capability.yaml v1.0

This directory contains the formal submission package for proposing `capability.yaml` as a hosted specification project under the Linux Foundation AI & Data (LF AI & Data) Foundation.

## Contents

| File | Description |
|------|-------------|
| `cover-letter.md` | Executive summary and proposal letter to LF AI & Data TAC |

| `README.md` | This file |

## Submission Checklist

- [x] Cover letter with problem statement and alignment rationale
- [x] Formal spec document (LF template format)
- [x] Proposed governance charter
- [x] IP and licensing declaration (Apache 2.0 / MIT)
- [x] Adoption metrics
- [ ] TAC nomination from an existing LF AI & Data member (required)
- [ ] Secondary co-editor from non-Capacium organization (recommended)
- [ ] LF project proposal form completed (external — https://lfai.foundation/projects/proposing-a-project/)
